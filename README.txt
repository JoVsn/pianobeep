To compile :
- Reach the root folder containing all the files in your terminal with g++.exe installed on your computer
- Type in the following command : g++ headers/*.hpp src/*.cpp main.cpp -o main

To execute the program :
- Reach the root folder containing all the files in your terminal
- Type in the following command : main.exe (or ./main if you are on Linux)

BE CAREFUL : 
The program may not work if you use Linux. Some functions are only supposed to work on a Windows OS
